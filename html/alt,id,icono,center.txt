  -------Alt, ID, iconos y center--------




no puede faltar la etiqueta alt cuando ponemos una imagen por eje;




   <!DOCTYPE html>
<html lang="en">
<head>
    <title>Document</title>
</head>
<body>
    <img src="steve.jpg" alt="steve" title="jugador">
</body>
</html>


despues tenemos la etiqueta center.
es para centrar algo,atexto,imagen,etc,

por eje:


   <!DOCTYPE html>
<html lang="en">
<head>
    <title>Document</title>
</head>
<body>
     <center>
    <img src="steve.jpg" alt="steve" title="jugador">
     </center>
</body>
</html>
