"name" en input es como un identificador

hay distintos tipos de input como por ejemplo:


input type="text" : hola 
input type="number" : 123
input type="password" : ****
input type="email" : ricardoalvarez21@gmail.com
input type="color" : rojo
input type="range" : es el rango de numeros por eje: min:1 max:10  del 1 al 10 si 
movemos la palanca para abajo nos da un 4 y si movemos para arriba un 8, y si lo
 ponemos al maximo sera un 10
input type="date" : 10/05/2023
input type="time" : 12:54pm
input type="button" un boton que tiene que tener si o si el "value" por ejemplo: value:boton
entonces el boton dirá "boton"
input type="submit": un boton que envia los datos
